# Contacter le support

## Objectif

Obtenir de l'aide du support Terrilink rapidement, avec le bon niveau d'information dès le premier message pour accélérer la résolution.

## Quand utiliser cette page

- Pour une question fonctionnelle non couverte par la documentation
- Pour signaler un bug bloquant
- Pour demander une fonctionnalité
- Pour toute question commerciale ou de facturation

## Canaux disponibles

| Canal | Délai de réponse | Idéal pour |
| --- | --- | --- |
| Email `support@terrilink.io` | 24h ouvrées | Questions générales, demandes fonctionnelles |
| Formulaire intégré (dans l'application) | 24h ouvrées | Bug avec capture d'écran |
| Téléphone (clients Premium / Enterprise) | 4h ouvrées | Incidents critiques |

## Informations à fournir

Pour accélérer la résolution, indiquez systématiquement :

1. **L'URL** où se produit le problème (`alumni.votre-ecole.fr/posts/123`)
2. **L'action** réalisée et le **résultat observé**
3. **Le résultat attendu**
4. **Une capture d'écran** ou une vidéo si possible
5. **Votre navigateur et votre OS** (Chrome 120 / macOS 14, Firefox 122 / Windows 11…)

## Étapes pour signaler depuis l'application

!!! info "Page en cours de rédaction"
    Le contenu détaillé sera ajouté en Phase 3.

## Résultat attendu

Un ticket support créé avec toutes les informations nécessaires, traité dans le délai correspondant à votre plan.

## Bonnes pratiques

!!! tip "Une demande, un ticket"
    Si vous avez plusieurs sujets, créez plusieurs tickets séparés. Cela permet un suivi clair et évite que des demandes secondaires soient perdues dans un échange long.

## Points d'attention

!!! warning "Données sensibles"
    Ne partagez jamais de mots de passe en clair dans un email au support. Si vous devez nous donner accès à un compte de test, créez un compte temporaire dédié et supprimez-le après résolution.
