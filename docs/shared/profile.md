# Mon profil

## Objectif

Personnaliser votre profil sur le réseau : photo, parcours, coordonnées, centres d'intérêt, paramètres de confidentialité.

## Quand utiliser cette fonctionnalité

- Après votre première connexion
- À chaque changement professionnel ou personnel (poste, ville, téléphone…)
- Pour ajuster ce que les autres membres voient de vous

## Points d'attention

!!! tip "Visibilité par défaut"
    À la création de votre profil, les informations de base (nom, prénom, promotion ou pays) sont visibles par tous les autres membres du réseau. Les informations sensibles (téléphone, email personnel) sont masquées par défaut.
