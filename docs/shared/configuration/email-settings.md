# Paramètres email

## Objectif

Configurer l'adresse d'expédition des emails envoyés par votre réseau (invitations, notifications, rappels d'événements) et les éléments de signature.

## Quand utiliser cette fonctionnalité

- À la configuration initiale du réseau
- Pour personnaliser l'expéditeur avec le nom de votre organisation (`reseau@votre-ecole.fr` plutôt que `no-reply@terrilink.io`)
- Lors d'un changement de domaine email institutionnel

## Étapes

!!! info "Page en cours de rédaction"

## Résultat attendu

Des emails envoyés avec votre adresse personnalisée, vos couleurs et votre signature.

## Bonnes pratiques

!!! tip "Tester la délivrabilité"
    Après configuration d'un domaine personnalisé, envoyez-vous un email test depuis la plateforme et vérifiez qu'il n'arrive pas en spam. Si c'est le cas, contactez le support pour finaliser la configuration SPF/DKIM.

## Points d'attention

!!! warning "Domaine personnalisé et délivrabilité"
    Utiliser un domaine personnalisé pour l'envoi nécessite la configuration d'enregistrements DNS spécifiques (SPF, DKIM, DMARC). Cette opération est réalisée par le support Terrilink avec l'aide de votre service informatique. Comptez 24 à 72h.
