# Modules activables

## Objectif

Activer ou désactiver les modules optionnels de votre réseau selon votre stratégie d'animation et votre plan d'abonnement.

## Quand utiliser cette fonctionnalité

- À la configuration initiale, pour ne présenter aux membres que ce qui est utile
- Au lancement d'une nouvelle phase d'animation (ex : activation du module mentorat)
- Pour passer à un plan supérieur et activer un module premium

## Modules disponibles

| Module | Disponibilité | Description courte |
| --- | --- | --- |
| Fil d'actualités | Standard | Publications, commentaires, réactions, ciblage |
| Messagerie privée | Standard | Échanges 1-to-1 et conversations de groupe |
| Cartographie | Standard | Carte interactive des membres |
| Événements | Standard | Création d'événements, inscriptions, rappels |
| Mentorat | Standard | Mise en relation mentor / mentee |
| Annuaire avancé | Standard | Filtres avancés, recherche par compétence |
| Module Carrière (recruteurs) | Premium | Espace recruteur, offres payantes, trust badge |
| Module Sponsors | Premium | Mise en avant de partenaires et opérations sponsorisées |
| Module Taxe d'apprentissage | Premium (alumni) | Campagne TA augmentée |

## Résultat attendu

Un réseau dont la complexité affichée correspond à votre maturité de pilotage et à votre plan.

## Bonnes pratiques

!!! tip "Activer progressivement"
    N'activez pas tous les modules d'un coup. Lancez d'abord le fil d'actualités et la messagerie, puis ajoutez les événements après quelques semaines, puis le mentorat. Une trop grande complexité initiale décourage les membres.

## Points d'attention

!!! warning "Désactivation et données"
    Désactiver un module ne supprime pas les données associées. Si vous désactivez puis réactivez le module Événements, vos événements passés réapparaissent intacts.
