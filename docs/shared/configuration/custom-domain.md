# Domaine personnalisé

## Objectif

Rendre votre réseau accessible via une URL aux couleurs de votre organisation (`alumni.votre-ecole.fr` plutôt que `votre-ecole.terrilink.io`).

## Quand utiliser cette fonctionnalité

- Au lancement officiel du réseau, pour une communication professionnelle
- Pour renforcer la confiance des membres (URL reconnaissable)
- Pour faciliter le référencement et la recherche en moteur

## Étapes

!!! info "Page en cours de rédaction"
    Le contenu détaillé sera ajouté en Phase 3.

## Résultat attendu

Votre réseau accessible depuis un sous-domaine de votre choix, avec un certificat SSL valide (HTTPS) émis automatiquement.

## Bonnes pratiques

!!! tip "Choisir un sous-domaine simple"
    Préférez `alumni.ecole.fr`, `reseau.association.fr`, `communaute.fondation.org`. Évitez les sous-domaines trop longs ou techniques.

## Points d'attention

!!! warning "Configuration DNS requise"
    L'activation d'un domaine personnalisé nécessite la création d'un enregistrement DNS (CNAME) côté votre fournisseur de domaine. La propagation peut prendre jusqu'à 48h. Le support Terrilink vous accompagne dans cette opération.

!!! warning "Certificat SSL"
    Le certificat SSL est émis automatiquement par notre infrastructure (Let's Encrypt) une fois la résolution DNS opérationnelle. Aucune action de votre part n'est requise.
