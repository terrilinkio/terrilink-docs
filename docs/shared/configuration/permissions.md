# Permissions et rôles

## Objectif

Configurer finement ce que chaque rôle (membre, modérateur, administrateur) peut faire sur votre réseau, dans la limite des autorisations standard.

## Quand utiliser cette fonctionnalité

- Pour restreindre le droit de publier aux administrateurs uniquement (réseau modéré strict)
- Pour autoriser les modérateurs à exporter la base de membres
- Pour ajuster la politique de modération à votre culture organisationnelle

## Points d'attention

!!! warning "Hiérarchie immuable"
    La hiérarchie des rôles (administrateur > modérateur > membre) reste fixe. Vous pouvez ajuster ce qu'un rôle donné est autorisé à faire, mais vous ne pouvez pas inverser la hiérarchie ni créer des rôles personnalisés en version 1.0.
