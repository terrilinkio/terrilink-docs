# Messagerie privée

## Objectif

Échanger en privé avec un ou plusieurs autres membres du réseau, sans passer par les emails personnels et sans quitter la plateforme.

## Quand utiliser cette fonctionnalité

- Pour reprendre contact avec un ancien camarade de promotion
- Pour échanger avec un mentor ou mentoré dans le cadre du module mentorat
- Pour préparer un événement entre plusieurs membres

## Points d'attention

!!! warning "Confidentialité des messages"
    Les administrateurs et le support Terrilink **ne lisent jamais** le contenu de vos messages privés. En revanche, en cas de signalement par votre interlocuteur, le contenu signalé pourra être examiné dans le cadre de la modération.
