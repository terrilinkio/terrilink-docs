# Documentation Terrilink

Bienvenue dans la documentation officielle de la plateforme **Terrilink** — l'infrastructure communautaire conçue pour les réseaux **alumni** et les **diasporas**.

Cette documentation s'adresse aux **administrateurs** d'une organisation cliente (école, association, fondation). Un guide membre destiné aux adhérents finaux est en préparation.

---

## Choisir votre parcours

<div class="grid cards" markdown>

-   :material-school: **Vous gérez un réseau alumni**

    ---

    Écoles, universités, grandes écoles, centres de formation.
    Cinq dimensions de ciblage : pays, programme, campus, promotion, type de membre.

    [:octicons-arrow-right-24: Démarrer en tant qu'administrateur alumni](alumni/admin/onboarding.md)

-   :material-earth: **Vous gérez un réseau diaspora**

    ---

    Associations régionales, fédérations expatriées, communautés territoriales.
    Ciblage par pays de résidence pour communications locales pertinentes.

    [:octicons-arrow-right-24: Démarrer en tant qu'administrateur diaspora](diaspora/admin/onboarding.md)

</div>
