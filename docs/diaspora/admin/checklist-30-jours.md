# Check-list des 30 premiers jours — Administrateur Diaspora

## Objectif

Guide pas-à-pas pour le premier mois d'un administrateur de réseau diaspora sur Terrilink. Vise à maximiser l'adoption par les adhérents et éviter les écueils classiques de démarrage.

## Quand utiliser cette fonctionnalité

- Après l'[onboarding administrateur](onboarding.md) initial
- Avant d'envoyer la première invitation massive aux adhérents
- Pour structurer la reprise d'administration d'un réseau existant
