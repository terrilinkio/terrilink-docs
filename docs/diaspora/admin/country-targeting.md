# Ciblage par pays de résidence

## Objectif

Diffuser une publication uniquement aux membres résidant dans un ou plusieurs pays, pour éviter d'inonder l'ensemble du réseau avec des annonces purement locales (soirée à Madrid, recommandation médecin à Londres…).

## Quand utiliser cette fonctionnalité

- Annonce d'un événement local (apéro, rencontre, sortie culturelle)
- Information pratique liée à un pays (changement de législation, démarche administrative)
- Recrutement d'un chapitre local

## Résultat attendu

Une publication visible uniquement par les membres dont le pays de résidence correspond à votre ciblage. Les administrateurs voient toutes les publications quel que soit leur pays.

## Bonnes pratiques

!!! tip "Conseil Terrilink"
    Les membres résidant dans un autre pays ne verront pas la publication. **Indiquez le pays dans le titre** si le post est volontairement local, pour clarifier au lecteur pourquoi il le voit (ex : « 🇪🇸 Apéro à Madrid le 23 mai »).

## Points d'attention

!!! warning "Saisie du pays côté membre"
    Le ciblage repose sur le pays renseigné par chaque membre dans son profil. Si vos adhérents n'ont pas renseigné ce champ, ils ne seront ciblés par aucune publication géographique.
