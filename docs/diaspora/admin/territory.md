# Territoire d'origine

## Objectif

Définir le territoire de référence de votre diaspora (région, pays, communauté). Ce paramètre conditionne le centre cartographique et permet d'orienter la communication autour d'un sentiment d'appartenance commun.

## Quand utiliser cette fonctionnalité

- À la configuration initiale du réseau
- En cas de rebranding ou de changement de périmètre territorial
- Pour ajuster la carte interactive et le centrage géographique
