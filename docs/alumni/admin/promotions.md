# Promotions

## Objectif

Gérer la liste des promotions (années de diplôme) de votre établissement, afin de permettre aux alumni de se rattacher à leur cohorte et d'offrir un ciblage précis lors des communications.

## Quand utiliser cette fonctionnalité

- Lors de la configuration initiale du réseau, avant le premier import d'alumni
- Chaque année scolaire, pour ajouter la nouvelle promotion sortante
- Quand vous lancez un module de mentorat (les promotions sont la base du matching)
