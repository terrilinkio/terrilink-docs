# Paramètres généraux

## Objectif

Configurer les paramètres globaux de votre organisation alumni : nom officiel, branding, domaine personnalisé, modules activés, politique d'abonnement.

## Quand utiliser cette fonctionnalité

- Lors de la mise en service initiale
- Après un changement d'identité visuelle de votre établissement
- Lorsqu'un module optionnel doit être activé ou désactivé

## Étapes

Ces paramètres sont organisés dans plusieurs pages dédiées :

- [Personnalisation et branding](../../shared/configuration/branding.md)
- [Modules activables](../../shared/configuration/modules.md)
- [Paramètres email](../../shared/configuration/email-settings.md)
- [Domaine personnalisé](../../shared/configuration/custom-domain.md)
- [Permissions et rôles](../../shared/configuration/permissions.md)

## Résultat attendu

Une configuration cohérente avec l'identité de votre établissement, prête à être présentée aux alumni.
