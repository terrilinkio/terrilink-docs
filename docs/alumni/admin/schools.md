# Écoles et programmes

## Objectif

Configurer le référentiel pédagogique de votre établissement : école de référence, programmes (filières, BTS, masters, certificats), campus en cas de multi-sites.

## Quand utiliser cette fonctionnalité

- À l'initialisation de votre réseau, avant le premier import d'alumni
- Lors d'une réforme pédagogique (nouvelle filière, fusion de cursus)
- Quand vous ajoutez un campus à votre établissement
